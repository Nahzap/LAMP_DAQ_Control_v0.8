## 📚 API PRINCIPAL: DAQController

### Inicialización

#### Constructor
```csharp
public DAQController(
    ILogger logger = null,
    IDeviceManager deviceManager = null,
    IProfileManager profileManager = null,
    IChannelManager channelManager = null,
    ISignalGenerator signalGenerator = null
)
```

**Ejemplo Básico:**
```csharp
// Inicialización simple
var controller = new DAQController();

// Con logger personalizado
var logger = new ConsoleLogger();

var controller = new DAQController(
    logger,
    deviceManager,
    profileManager,
    channelManager,
    signalGenerator
);
```

#### Initialize()
```csharp
public void Initialize(string profileName = null, int deviceNumber = 0)
```

**Parámetros:**
- `profileName`: Nombre del perfil XML (sin extensión)
- `deviceNumber`: Board ID del dispositivo (0-7)

**Excepciones:**
- `DAQInitializationException`: Error al inicializar dispositivo
- `ObjectDisposedException`: Controlador ya liberado

**Ejemplos:**
```csharp
// Inicializar PCIe-1824 (Board ID = 0)
controller.Initialize("PCIe1824_prof_v1", 0);

// Inicializar PCI-1735U (Board ID = 1)
controller.Initialize("PCI1735U_prof_v1", 1);

// Inicializar sin perfil (usa configuración por defecto)
controller.Initialize(null, 0);

// Con manejo de errores
try
{
    controller.Initialize("PCIe1824_prof_v1", 0);
    Console.WriteLine("Dispositivo inicializado correctamente");
}
catch (DAQInitializationException ex)
{
    Console.WriteLine($"Error: {ex.Message}");
}
```

### Control Analógico (PCIe-1824)

#### WriteVoltage()
```csharp
public void WriteVoltage(int channel, double value)
```

**Parámetros:**
- `channel`: Número de canal (0-31)
- `value`: Voltaje en V (0.0-10.0)

**Excepciones:**
- `ArgumentOutOfRangeException`: Canal fuera de rango
- `DAQOperationException`: Error de escritura

**Ejemplos:**
```csharp
// Escribir 5V en canal 0
controller.WriteVoltage(0, 5.0);

// Escribir 0V en todos los canales
for (int i = 0; i < 32; i++)
{
    controller.WriteVoltage(i, 0.0);
}

// Rampa manual
for (double v = 0.0; v <= 10.0; v += 0.1)
{
    controller.WriteVoltage(0, v);
    Thread.Sleep(10); // 10ms por paso
}

// Con validación
int channel = 0;
double voltage = 7.5;

if (channel >= 0 && channel < controller.ChannelCount)
{
    if (voltage >= 0.0 && voltage <= 10.0)
    {
        controller.WriteVoltage(channel, voltage);
    }
}
```

#### RampChannelValue()
```csharp
public Task RampChannelValue(int channel, double targetValue, int durationMs)
```

**Parámetros:**
- `channel`: Número de canal (0-31)
- `targetValue`: Voltaje objetivo (0.0-10.0)
- `durationMs`: Duración de la rampa en ms

**Retorna:** Task asíncrono

**Ejemplos:**
```csharp
// Rampa de 0V a 10V en 1 segundo

// Rampa de 10V a 0V en 500ms
await controller.RampChannelValue(0, 0.0, 500);

// Rampa con callback
controller.RampChannelValue(0, 5.0, 2000)
    .ContinueWith(task => 
    {
        Console.WriteLine("Rampa completada");
    });

// Múltiples rampas secuenciales
await controller.RampChannelValue(0, 5.0, 1000);
await Task.Delay(500);
await controller.RampChannelValue(0, 0.0, 1000);

// Múltiples canales en paralelo
var tasks = new List<Task>();
for (int ch = 0; ch < 4; ch++)
{
    tasks.Add(controller.RampChannelValue(ch, 10.0, 1000));
}
await Task.WhenAll(tasks);
```

#### StartSignalGeneration()
```csharp
public void StartSignalGeneration(
    int channel,
    double frequency,
    double amplitude,
    double offset
)
```

**Parámetros:**
- `channel`: Número de canal (0-31)
- `frequency`: Frecuencia en Hz (0.1-10000)
- `amplitude`: Amplitud en V (0.0-10.0)
- `offset`: Offset en V (0.0-10.0)

**Fórmula:** `V(t) = offset + amplitude * sin(2π * frequency * t)`

**Ejemplos:**
```csharp
// Señal senoidal 1Hz, 5V amplitud, 5V offset
controller.StartSignalGeneration(0, 1.0, 5.0, 5.0);
// Resultado: Oscila entre 0V y 10V

// Señal 100Hz, 2V amplitud, 5V offset
controller.StartSignalGeneration(0, 100.0, 2.0, 5.0);
// Resultado: Oscila entre 3V y 7V

// Señal de baja frecuencia (0.5Hz)
controller.StartSignalGeneration(0, 0.5, 5.0, 5.0);
// Resultado: 1 ciclo cada 2 segundos

// Múltiples canales con diferentes frecuencias
controller.StartSignalGeneration(0, 1.0, 5.0, 5.0);   // 1Hz
controller.StartSignalGeneration(1, 10.0, 3.0, 5.0);  // 10Hz
controller.StartSignalGeneration(2, 100.0, 2.0, 5.0); // 100Hz

// Esperar y detener
Thread.Sleep(5000);
controller.StopSignalGeneration();
```

#### ReadDigitalBit()
```csharp
public bool ReadDigitalBit(int port, int bit)
```

**Parámetros:**
- `port`: Número de puerto (0-3)
- `bit`: Número de bit (0-7)

**Retorna:** true=HIGH, false=LOW

**Ejemplos:**
```csharp
// Leer bit específico
bool state = controller.ReadDigitalBit(0, 0);
Console.WriteLine($"Bit 0: {(state ? "HIGH" : "LOW")}");

// Leer todos los bits de un puerto
for (int bit = 0; bit < 8; bit++)
{
    bool state = controller.ReadDigitalBit(0, bit);
    Console.WriteLine($"Bit {bit}: {state}");
}

// Esperar a que un bit cambie
while (!controller.ReadDigitalBit(0, 0))
{
    Thread.Sleep(10);
}
Console.WriteLine("Bit 0 activado!");

// Detectar flanco ascendente
bool lastState = false;
while (true)
{
    bool currentState = controller.ReadDigitalBit(0, 0);
    if (currentState && !lastState)
    {
        Console.WriteLine("Flanco ascendente detectado");
    }
    lastState = currentState;
    Thread.Sleep(10);
}
```

### StartMonitoring()
```csharp
public void StartMonitoring(int deviceNumber, int intervalMs = 50)
```

**Parámetros:**
- `deviceNumber`: Board ID del dispositivo
- `intervalMs`: Intervalo de lectura en ms (10-1000)

**Eventos:**
- `DataReceived`: Se dispara en cada lectura
- `ErrorOccurred`: Se dispara en caso de error

**Ejemplos:**
```csharp
// Monitoreo básico
var monitor = new DigitalInputMonitor();

monitor.DataReceived += (sender, e) =>
{
    Console.WriteLine($"Puerto 0: {e.PortData[0]:X2}");
    Console.WriteLine($"Puerto 1: {e.PortData[1]:X2}");
    Console.WriteLine($"Puerto 2: {e.PortData[2]:X2}");
    Console.WriteLine($"Puerto 3: {e.PortData[3]:X2}");
};

monitor.StartMonitoring(1, 100); // 100ms = 10Hz

// Monitoreo con detección de cambios
monitor.DataReceived += (sender, e) =>
{
    if (e.HasChanged)
    {
        Console.WriteLine($"Cambio detectado en {e.Timestamp}");
    }
};

// Monitoreo de bit específico
monitor.DataReceived += (sender, e) =>
{
    bool bit0 = e.GetBit(0, 0);
    if (bit0)
    {
        Console.WriteLine("Bit 0 del puerto 0 está HIGH");
    }
};

// Monitoreo con manejo de errores
monitor.ErrorOccurred += (sender, e) =>
{
    Console.WriteLine($"Error: {e.GetException().Message}");
};

monitor.StartMonitoring(1, 50);
```

## 📚 EJEMPLOS COMPLETOS

### Ejemplo 1: Sistema de Control de Temperatura

```csharp
using System;
using System.Threading.Tasks;
using LAMP_DAQ_Control_v0_8.Core.DAQ;

class TemperatureController
{
    private DAQController _daq;
    private const int HEATER_CHANNEL = 0;
    private const double TARGET_TEMP = 50.0; // °C
    
    public async Task Run()
    {
        _daq = new DAQController();
        _daq.Initialize("PCIe1824_prof_v1", 0);
        
        // PID control loop
        double kp = 0.5, ki = 0.1, kd = 0.05;
        double integral = 0, lastError = 0;
        
        while (true)
        {
            double currentTemp = ReadTemperature();
            double error = TARGET_TEMP - currentTemp;
            
            integral += error * 0.1; // dt = 100ms
            double derivative = (error - lastError) / 0.1;
            
            double output = kp * error + ki * integral + kd * derivative;
            output = Math.Max(0, Math.Min(10, output)); // Clamp 0-10V
            
            _daq.WriteVoltage(HEATER_CHANNEL, output);
            
            Console.WriteLine($"Temp: {currentTemp:F2}°C, Output: {output:F2}V");
            
            lastError = error;
            await Task.Delay(100);
        }
    }
    
    private double ReadTemperature()
    {
        // Leer sensor de temperatura (simulado)
        return 25.0 + new Random().NextDouble() * 5.0;
    }
}
```

### Ejemplo 3: Monitor de Entradas Digitales

```csharp
class DigitalMonitor
{
    private DigitalInputMonitor _monitor;
    private int[] _counters = new int[32]; // Contadores por bit
    
    public void StartMonitoring()
    {
        _monitor = new DigitalInputMonitor();
        
        _monitor.DataReceived += OnDataReceived;
        _monitor.ErrorOccurred += OnError;
        
        _monitor.StartMonitoring(1, 100); // 10Hz
        
        Console.WriteLine("Monitoreando entradas digitales...");
        Console.ReadLine();
        
        _monitor.StopMonitoring();
        PrintStatistics();
    }
    
    private void OnDataReceived(object sender, DigitalDataEventArgs e)
    {
        // Contar transiciones
        for (int port = 0; port < 4; port++)
        {
            for (int bit = 0; bit < 8; bit++)
            {
                if (e.GetBit(port, bit))
                {
                    _counters[port * 8 + bit]++;
                }
            }
        }
        
        // Mostrar estado cada segundo
        if (e.Timestamp.Millisecond < 100)
        {
            Console.WriteLine($"[{e.Timestamp:HH:mm:ss}] " +
                $"P0:{e.PortData[0]:X2} " +
                $"P1:{e.PortData[1]:X2} " +
                $"P2:{e.PortData[2]:X2} " +
                $"P3:{e.PortData[3]:X2}");
        }
    }
    
    private void OnError(object sender, ErrorEventArgs e)
    {
        Console.WriteLine($"ERROR: {e.GetException().Message}");
    }
    
    private void PrintStatistics()
    {
        Console.WriteLine("\n=== Estadísticas ===");
        for (int i = 0; i < 32; i++)
        {
            Console.WriteLine($"Bit {i}: {_counters[i]} activaciones");
        }
    }
}
```

### Ejemplo 4: Sistema de Alarma

```csharp
class AlarmSystem
{
    private DAQController _daq;
    private DigitalInputMonitor _monitor;
    private const int SIREN_CHANNEL = 0;
    
    public void Run()
    {
        _daq = new DAQController();
        _daq.Initialize("PCIe1824_prof_v1", 0);
        
        _monitor = new DigitalInputMonitor();
        _monitor.DataReceived += CheckSensors;
        _monitor.StartMonitoring(1, 50);
        
        Console.WriteLine("Sistema de alarma activo");
        Console.ReadLine();
        
        _monitor.StopMonitoring();
        _daq.Dispose();
    }
    
    private void CheckSensors(object sender, DigitalDataEventArgs e)
    {
        // Sensor de puerta: Puerto 0, Bit 0
        bool doorOpen = e.GetBit(0, 0);
        
        // Sensor de movimiento: Puerto 0, Bit 1
        bool motionDetected = e.GetBit(0, 1);
        
        // Sensor de humo: Puerto 0, Bit 2
        bool smokeDetected = e.GetBit(0, 2);
        
        if (doorOpen || motionDetected || smokeDetected)
        {
            ActivateAlarm();
            LogEvent($"Alarma: Door={doorOpen}, Motion={motionDetected}, Smoke={smokeDetected}");
        }
        else
        {
            DeactivateAlarm();
        }
    }
    
    private void ActivateAlarm()
    {
        // Generar señal de sirena (1Hz pulsante)
        _daq.StartSignalGeneration(SIREN_CHANNEL, 1.0, 5.0, 5.0);
    }
    
    private void DeactivateAlarm()
    {
        _daq.StopSignalGeneration();
        _daq.WriteVoltage(SIREN_CHANNEL, 0.0);
    }
    
    private void LogEvent(string message)
    {
        string timestamp = DateTime.Now.ToString("yyyy-MM-dd HH:mm:ss.fff");
        Console.WriteLine($"[{timestamp}] {message}");
    }
}
```

### Ejemplo 5: Secuenciador de Pasos

```csharp
class StepSequencer
{
    private DAQController _daq;
    
    public async Task RunSequence()
    {
        _daq = new DAQController();
        _daq.Initialize("PCIe1824_prof_v1", 0);
        
        var steps = new[]
        {
            new { Channel = 0, Voltage = 5.0, Duration = 1000 },
            new { Channel = 1, Voltage = 7.5, Duration = 1500 },
            new { Channel = 2, Voltage = 3.0, Duration = 2000 },
            new { Channel = 3, Voltage = 10.0, Duration = 1000 },
        };
        
        Console.WriteLine("Ejecutando secuencia...");
        
        foreach (var step in steps)
        {
            Console.WriteLine($"Paso: Canal {step.Channel} = {step.Voltage}V por {step.Duration}ms");
            
            await _daq.RampChannelValue(step.Channel, step.Voltage, 500);
            await Task.Delay(step.Duration);
            await _daq.RampChannelValue(step.Channel, 0.0, 500);
        }
        
        Console.WriteLine("Secuencia completada");
        _daq.ResetAllChannels();
    }
}
```
