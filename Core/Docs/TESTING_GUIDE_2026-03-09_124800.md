## 🔍 SISTEMA DE LOGGING

### Ubicación de Logs

Los logs se guardan automáticamente en:
```
C:\Users\[Usuario]\AppData\Roaming\LAMP_DAQ_Control\Logs\
```

Formato de archivo:
```
LAMP_DAQ_yyyyMMdd_HHmmss.log
```

### Tipos de Logs Registrados

#### 1. Acciones de Usuario
```
[USER ACTION] SetDc | Details: Channel=0, Voltage=5.0V
[BUTTON CLICK] GenerateRamp | ViewModel: AnalogControlViewModel
[VALUE CHANGE] Voltage: 0 → 5.0 | Source: AnalogControlViewModel
```

#### 2. Operaciones de Hardware
```
[ANALOG WRITE] Channel=0, Voltage=5.000V | Status: SUCCESS
[DIGITAL WRITE] Port=0, Value=0xFF | Status: SUCCESS
[DIGITAL READ] Port=0, Value=0xAA | Status: SUCCESS
```

#### 3. Generación de Señales
```
[SIGNAL START] Channel=0, Freq=100Hz, Amp=5.00V, Offset=5.00V
[SIGNAL STOP] Channel=0
[RAMP START] Channel=0, 0.00V → 10.00V, Duration=1000ms
[RAMP END] Channel=0, FinalValue=10.00V, ActualDuration=1002ms
```

#### 4. Detección e Inicialización
```
[DEVICE DETECTION] Found 2 device(s)
[DEVICE INIT] PCIe-1824 (ID=0) | Status: SUCCESS
[TIMING] Device Detection completed in 45ms
```

#### 5. Errores y Excepciones
```
[EXCEPTION] Context: SetDc
  Exception: DAQOperationException
  Message: Channel out of range
  StackTrace: ...
```

### Características del Sistema de Logging

✅ **Rotación Automática**: Los archivos se rotan al alcanzar 10 MB  
✅ **Limpieza Automática**: Logs antiguos (>30 días) se eliminan  
✅ **Thread-Safe**: Seguro para operaciones concurrentes  
✅ **Timestamps Precisos**: Formato `yyyy-MM-dd HH:mm:ss.fff`  
✅ **Múltiples Destinos**: Console + File + UI simultáneamente

## 🎯 VALIDACIONES IMPLEMENTADAS

### Validación de Parámetros Analógicos

```csharp
// Canal válido: 0-31
if (channel < 0 || channel >= ChannelCount)
    throw new ArgumentOutOfRangeException(nameof(channel));

// Voltaje válido: 0-10V
if (voltage < 0.0 || voltage > 10.0)
    throw new ArgumentOutOfRangeException(nameof(voltage));

// Frecuencia válida: > 0
if (frequency <= 0)
    throw new ArgumentOutOfRangeException(nameof(frequency));

// Amplitud válida: 0-10V
if (amplitude < 0 || amplitude > 10)
    throw new ArgumentOutOfRangeException(nameof(amplitude));

// Offset válido: 0-10V
if (offset < 0 || offset > 10)
    throw new ArgumentOutOfRangeException(nameof(offset));

// Amplitud + Offset no debe exceder 10V
if (amplitude + offset > 10)
    throw new ArgumentException("Amplitude + Offset cannot exceed 10V");
```

### Validación de Parámetros Digitales

```csharp
// Puerto válido: 0-3
if (port < 0 || port > 3)
    throw new ArgumentOutOfRangeException(nameof(port));

// Bit válido: 0-7
if (bit < 0 || bit > 7)
    throw new ArgumentOutOfRangeException(nameof(bit));
```

## 🔧 TROUBLESHOOTING

### Problema: Tests no se descubren en Test Explorer

**Solución:**
1. Rebuild del proyecto de tests
2. Cerrar y reabrir Visual Studio
3. Limpiar cache: `dotnet clean`

### Problema: Error al restaurar paquetes NuGet

**Solución:**
```powershell
cd C:\LAMP_CONTROL\LAMP_DAQ_Control_v0.8.Tests
nuget restore packages.config
```

### Problema: Logs no se generan

**Verificar:**
1. Permisos de escritura en `%APPDATA%\LAMP_DAQ_Control\Logs`
2. Espacio en disco disponible
3. Revisar excepciones en consola
